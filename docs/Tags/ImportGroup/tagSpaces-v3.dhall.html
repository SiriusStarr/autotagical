Tags.ImportGroup.tagSpaces-v3

Function

Given JSON in Text form and the name of the group to import as Text, create a tag group for use with the hasGroup predicate from TagSpaces JSON.

Usage

let Autotagical = https://siriusstarr.github.io/autotagical/dhall/package.dhall

in  Autotagical.Tags.ImportGroup.tagSpaces-v3
      ./Path/To/TagSpaces-v3.json as Text
      "GroupName"
