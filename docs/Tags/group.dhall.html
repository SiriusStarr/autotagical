Tags.group

Function

Given a list of tags, create a tag group for use with the hasGroup predicate. Groups may also be imported using the functions found in Tags.ImportGroup.

Usage

let Autotagical = https://siriusstarr.github.io/autotagical/dhall/package.dhall

in  Autotagical.Tags.group
      [ Autotagical.Tags.tag "tag1", Autotagical.Tags.tag "tag2" ]
