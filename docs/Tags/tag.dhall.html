Tags.tag

Function

Given text, create a tag (for the purposes of creating a tag group, for the hasTag predicate, etc.). Note that the text will match the tag part of a tagged value, or the entirety of a non-tagged value.

Usage

let Autotagical = https://siriusstarr.github.io/autotagical/dhall/package.dhall

in  Autotagical.Tags.tag "tag1"
