Renaming.Name.Component.originalName

Component

Evaluates to the complete original file name (but not extension or tags) of an input file. Useful if you want to preserve original file names but also add to them in some fashion.

Example

File Name: "document [tag1 tag2].pdf"

Name Template: [ originalName, text "renamed" ]

Output: "document renamed"

Usage

let Autotagical = https://siriusstarr.github.io/autotagical/dhall/package.dhall

in  Autotagical.Renaming.Name.Component.originalName

    : Component
    = λ(Component : Type) →
        let ComponentF =
              < FileDuplicateNumberF
