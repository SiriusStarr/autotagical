Renaming.Name.Component.Interpret.numberAsYear

Component

Given a year format, a natural number representing the most "modern" year to be considered (relevant for 1- or 2-digit numbers), and a component, attempt to interpret the component as a number and then output it as a year in the requested format.

Examples

  • Format: 1999

    Latest Year: 2020

    Component: "23"

    Output: "1923"

  • Format: 1999

    Latest Year: 2020

    Component: "12"

    Output: "2012"

  • Format: 99

    Latest Year: 2020

    Component: "23"

    Output: "23"

Usage

let Autotagical = https://siriusstarr.github.io/autotagical/dhall/package.dhall

in  Autotagical.Renaming.Name.Component.Interpret.numberAsYear
      Autotagical.Renaming.Name.Component.Interpret.YearFormat.`99`
      2020
      (Autotagical.Renaming.Name.Component.text "03")
