Renaming.Name.Component.Interpret.YearFormat

Formats

Specifies the format to write a year as for Renaming.Name.Component.Interpret.numberAsYear.

  • 99:

    4 -> 04

    1843 -> 43

  • 1999:

    4 -> 2004 (Depending on latest year passed to numberAsYear)

    1754 -> 1754
