Config.With.LogLevel.error

Default Behavior

By default, autotagical only reports warnings, which are unexpected situations or cases where files could not be sorted, moved, renamed, etc.

Option Behavior

This option sets logging level to Error. This isn't recommended, unless you know warnings are going to be thrown and you're fine with ignoring that.
