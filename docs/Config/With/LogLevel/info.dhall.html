Config.With.LogLevel.info

Default Behavior

By default, autotagical only reports warnings, which are unexpected situations or cases where files could not be sorted, moved, renamed, etc.

Option Behavior

This option set logging level to Info. This is useful if you're trying to debug your Predicates, InputFormat, etc. It's often helpful to combine this with Config.With.dryRun, if you'd like to check out the behavior of your settings without actually manipulating files.
