Input.With.TrailingTagSeparator.forbidden

Default Behavior

By default, trailing tag separators are optional. For example, [tag1,tag2,] and [tag1,tag2] are both valid, if the tag separator is ,

Option Behavior

This option sets the input format to be more strict and instead specifically forbid trailing tag separators. For example, [tag1,tag2,] would fail to parse in the above example.
