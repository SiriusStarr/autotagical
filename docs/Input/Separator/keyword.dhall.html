Input.Separator.keyword

Function

Given nonempty Text, create a separator that matches that literal text. For example, Input.Separator.keyword "[" will match the literal character [, and Input.Separator.keyword "|betweentags|" would parse tag1|betweentags|tag2 as [tag1, tag2].
